### Run Locally

Clone the project

```bash
  git clone https://github.com/alanalexxis/tvmaze-search.git
```

Go to the project directory

```bash
  cd tvmaze-search
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```

### Click to open webapp in browser

http://localhost:5173/
